===========================
How to Use 3-say_my_name.py
===========================

This module defines a function ``say_my_name(first_name, last_name="")`` to print a formatted name.

Usage
=====

The function ``say_my_name(first_name, last_name="")`` takes two string arguments: ``first_name`` and ``last_name``. It prints a formatted string that says "My name is <first_name> <last_name>". If the ``last_name`` argument is not provided, it defaults to an empty string.

Example Usage:
--------------

Here's an example of how to use the ``say_my_name()`` function:

```python
>>> say_my_name("John", "Doe")
My name is John Doe

>>> say_my_name("Alice")
My name is Alice
